
🚀 Pratik – The Open Tech Stack & Company Logo API
A community-powered logo hub for developers.

https://i.imgflip.com/92f0u8.jpg

🌐 What is Pratik?
Pratik is an open-source, community-driven API that provides tech stack logos, company logos, and brand icons with one simple endpoint:

👉 https://pratik.devloke.com
Use it anywhere:

Web apps
Mobile apps
Dashboards
Developer tools
Blogs & documentation
And contribute new logos anytime.

🧩 Why Pratik?
Because every developer has been through:

Searching “Next.js logo transparent PNG”
Finding pixelated, ugly, or watermarked images
SVGs hidden behind infinite modals
Inconsistent assets in production
Pratik fixes this.
Clean. Consistent. Developer-first.

📦 Features
✔️ Clean SVG & PNG logos
✔️ Instant API access
✔️ Free & open-source
✔️ Community contributions
✔️ Standardized logo structure
✔️ Easy integration everywhere
🔥 API Usage
Get a logo
https://pratik.devloke.com/logo/:name
Example:

https://pratik.devloke.com/logo/react
Get PNG format
https://pratik.devloke.com/logo/react?format=png
List all logos
https://pratik.devloke.com/logos
Search logos
https://pratik.devloke.com/search?query=cloud

🤝 How to Contribute
We ❤️ contributors.

1. Fork the repo
git fork https://github.com/your-org/pratik
2. Add your logo
Create a folder inside /logos/<brand-name>/
Add svg and png versions
Make sure they are clean & unmodified
3. Submit a Pull Request
Include:

Brand name
Official source link
Clean optimized SVG
